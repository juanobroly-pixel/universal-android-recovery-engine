# Privacy Policy

## Data Collection
Universal Android Recovery Engine collects minimal data required for its operation:
- **Authentication Data**: Managed via Firebase Auth.
- **Query Logs**: Stored to improve natural language processing (optional).

## Data Usage
- Data is used solely to provide the services requested by the user.
- System commands are executed locally and are not transmitted to external servers except for logging purposes if enabled.

## Third-Party Services
- **Google Cloud Platform**: For BigQuery and IAM.
- **Firebase**: For authentication and database management.

## User Rights
Users have the right to access, correct, or delete their data at any time via the application settings.
