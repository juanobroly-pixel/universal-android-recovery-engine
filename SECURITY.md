
Política de Privacidad - Mi Aplicación AUT

**Transparencia de Código Abierto**
Como proyecto de código abierto, cualquier usuario puede auditar nuestro código para verificar el manejo de datos técnicos.

**Datos Recolectados**
Solo recolectamos información técnica necesaria para el rescate del sistema:
* Modelo del dispositivo (ej. SM-A346M).
* Nivel de binario actual.
* Estado del bootloader.

**Uso de Datos**
Esta información se usa únicamente para descargar el software oficial correcto y evitar daños al hardware. No compartimos información con terceros ajenos a los servicios de actualización (Google Cloud/Firebase)
